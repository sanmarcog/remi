# Remi 💕

A cute pastel card game for two — built for Guille & Drew.

- **[index.html](index.html)** — online Remi: one deck, no jokers, 7 cards,
  display-as-you-go melds. One of you starts a table, shares the code, and you
  play live from anywhere (moves sync through a public MQTT relay and wait
  patiently if someone's offline).
- **[couch.html](couch.html)** — the pass-and-play edition for when you're on
  the same sofa.

Add your photos from ⚙️ settings on each phone — they show up in the polaroids.
